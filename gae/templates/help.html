Solar System ID
The ID number of the solar system of origin
Corporation ID
The ID number of corporation owning the stations of interest
Security Mask
An integer representing a bit field. Each set bit flags a system security rating for avoidence when calculating paths. There are 11 bits ordered by system security rating with zero sec in the least significant bit. For example, 0b00000111111 is decimal 63 and would set avoidance for any sectors with a security rating at or below 0.5.
